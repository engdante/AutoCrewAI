# Tasks for Crypto Market Analysis

## Task 1: Gather Latest News and Prices (Crypto Researcher)
- **Objective**: Collect the latest news and prices of top 10 cryptocurrencies by market cap using web search.
- **Input**: None (uses web search).
- **Output**: A detailed report containing the latest news, prices, and key metrics (e.g., market cap, volume) for the top 10 cryptocurrencies.

## Task 2: Analyze Data (Market Analyst)
- **Objective**: Analyze the gathered data to identify trends, patterns, and key insights about the top 10 cryptocurrencies by market cap.
- **Input**: The report generated by the Crypto Researcher.
- **Output**: An analysis report highlighting trends, significant price changes, market sentiment, and other relevant insights.

## Task 3: Compile Final Report (Report Compiler)
- **Objective**: Compile the analysis into a final report that is clear, concise, and well-structured for stakeholders.
- **Input**: The analysis report generated by the Market Analyst.
- **Output**: A polished final report summarizing the latest news and prices of top 10 cryptocurrencies by market cap, along with insights and trends.
