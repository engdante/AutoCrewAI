## üìñ Overview

### What Are Tools?

Tools are external capabilities that CrewAI agents can use to perform operations beyond text generation:
- üîç **Search Tools**: Web search, website scraping
- üìÑ **File Tools**: Read/write files, process documents (PDF, DOCX, CSV, etc.)
- üíª **Code Tools**: Execute Python code, interpret scripts
- üìä **Data Tools**: Analyze data, search GitHub, YouTube, etc.

### Goal

Enable agents to automatically use tools based on:
1. **Tools specified in Crew.md** (manual assignment)
2. **Auto-generated by Generate agent** (intelligent selection)
3. **Executed during "Run Crew"** (automatic tool invocation)

```python
def load_tools_for_agent(tools_string):
    """
    Load tools from comma-separated string of tool names.
    
    Args:
        tools_string: String like "FileReadTool, WebsiteSearchTool, SerperDevTool"
    
    Returns:
        List of instantiated tool objects
    """
    if not tools_string or tools_string.strip() == "":
        return []
    
    # Import available tools
    from crewai_tools import (
        FileReadTool,
        WebsiteSearchTool,
        SerperDevTool,
        DirectoryReadTool,
        PDFSearchTool,
        DOCXSearchTool,
        CSVSearchTool,
        TXTSearchTool,
        JSONSearchTool,
        # Add more as needed
    )
    
    # Map tool names to classes
    tool_map = {
        'FileReadTool': FileReadTool,
        'WebsiteSearchTool': WebsiteSearchTool,
        'SerperDevTool': SerperDevTool,
        'DirectoryReadTool': DirectoryReadTool,
        'PDFSearchTool': PDFSearchTool,
        'DOCXSearchTool': DOCXSearchTool,
        'CSVSearchTool': CSVSearchTool,
        'TXTSearchTool': TXTSearchTool,
        'JSONSearchTool': JSONSearchTool,
        # Add more mappings as needed
    }
    
    tools_list = []
    tool_names = [t.strip() for t in tools_string.split(',')]
    
    for tool_name in tool_names:
        if tool_name in tool_map:
            try:
                # Instantiate the tool
                tool_instance = tool_map[tool_name]()
                tools_list.append(tool_instance)
                print(f"  ‚úÖ Loaded tool: {tool_name}")
            except Exception as e:
                print(f"  ‚ùå Failed to load {tool_name}: {e}")
        else:
            print(f"  ‚ö†Ô∏è Unknown tool: {tool_name}")
    
    return tools_list
```

## üêõ Troubleshooting

### Common Issues

#### Issue 1: "Tool not found" error

**Symptom:**
```
‚ö†Ô∏è Unknown tool: SerperDevTool
```

**Cause:** Tool not in tool_map in `load_tools_for_agent()`

**Solution:** Add tool to import and tool_map:
```python
from crewai_tools import SerperDevTool
tool_map = {
    'SerperDevTool': SerperDevTool,
    # ...
}
```

#### Issue 2: "Module not found: crewai_tools"

**Symptom:**
```
ModuleNotFoundError: No module named 'crewai_tools'
```

**Cause:** crewai-tools package not installed

**Solution:**
```bash
pip install crewai-tools
```

#### Issue 3: Tool loads but doesn't execute

**Symptom:** Tool appears in agent but never gets used

**Cause:** Agent doesn't know when to use tool

**Solution:** Make task description more explicit:
```markdown
**Description**: Use WebsiteSearchTool to search for information about {task_input}
```

#### Issue 4: API Key errors

**Symptom:**
```
Error: SERPER_API_KEY not found
```

**Cause:** Tool requires API key that's not set

**Solution:** Add to .env file:
```bash
SERPER_API_KEY=your_key_here
```

Or use only confirmed free tools (FileReadTool, WebsiteSearchTool)

Add debug prints to verify tools are loaded:
